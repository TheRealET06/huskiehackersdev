Pizza Hack - Cloud Edition

🎨 Air Drawing, Now on the Cloud!

An Interactive Cloud & Networking Project for Pizza Hack

Ever imagined drawing in the air with a friend in real time, from different locations? In this Cloud-powered version of our Air Drawing project, you'll take your skills to the next level by hosting a shared virtual canvas online! Multiple users can draw in the air, and their drawings will appear on each other’s screens—live!

🔍 How It Works

  • Track Movements: Use a webcam + OpenCV + MediaPipe to capture motion.
  • Cloud-Powered Canvas: Sync drawings across multiple users using Firebase Realtime Database or WebSockets.
  • Live Collaboration: Draw together in real-time from different devices.
  • Erase Feature: Close your hand into a fist to clear the canvas.
  • Challenge Mode: Compete by drawing shapes or writing “Huskie Hackers”!

🔧 Tools You’ll Need

  • ✅ Python
  • ✅ OpenCV
  • ✅ MediaPipe
  • ✅ NumPy
  • ✅ Firebase/WebSockets

🚀 Getting Started

1. Install Dependencies:

pip install opencv-python mediapipe numpy flask-socketio firebase-admin

2. Set Up Your Webcam with OpenCV:

import cv2
cap = cv2.VideoCapture(0)  # Access your webcam

3. Use MediaPipe to Track Hand Movements.

4. Sync Drawings Across Users with Firebase/WebSockets.

5. Draw Together in Real-Time from Any System!

✨ Bonus Features (Optional for Enthusiasts!)

  • 🔥 User Authentication: Login to save your drawings.
  • 🔥 Room System: Join specific drawing rooms with friends.
  • 🔥 Multi-Color Support: Assign different colors to different users.
  • 🔥 Gesture Shortcuts: Use gestures to change tools (e.g., two fingers for color change).
