April Schedule

The Huskie Hackers program, by the Office of Innovation at Northern Illinois University, is a unique initiative to support and develop student talent through active participation in the Major League Hacking (MLH) Hackathon season.

Week 1 (April 1 - 4): Introduction to APIs

Wednesday, April 2 (3 PM - 5 PM):

  • Introduction to APIs and how they work.
  • Explore real-world examples.
  • Use Postman to send requests to public APIs.

Thursday, April 3 (3 PM - 5 PM):

  • Learn how to fetch data using APIs in Python and JavaScript.
  • Make a GET request to a public API and display results.
  • Work in teams to modify the request (change parameters, explore responses).

Friday, April 4 (2 PM - 4 PM):

  • Choose an API from a list.
  • Write a script to fetch and display data in a creative way.
  • Present findings in 5 minutes.

Week 2 (April 7 - 11): Authentication & Building API-Powered Apps

Wednesday, April 9 (3 PM - 5 PM):

  • Understanding API keys and OAuth.
  • Use an API that requires authentication.
  • Work in pairs to fetch data and share results.

Thursday, April 10 (3 PM - 5 PM):

  • Use HTML + JavaScript (or Python Flask) to display API data.
  • Fetch data from an API and show it on a webpage.
  • Work in teams to make a small, functional project.

Friday, April 11 (2 PM - 4 PM):

  • Pick any API and build something cool (chatbot, dashboard, fun app).
  • Work independently or in small teams.
  • Present your project in 5 minutes.

Week 3 (April 14 - 18): Global Hack Week [OPTIONAL]

Online: Spend a week building awesome API-based hacks. Boost your rank!
